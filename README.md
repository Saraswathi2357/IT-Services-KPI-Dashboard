# IT Services KPI Dashboard

## Project Overview
This project focuses on analyzing IT services performance using SQL, Python, Power BI, and Excel VBA. The objective is to transform raw operational data into meaningful business insights related to project delivery, resource utilization, revenue, and client satisfaction.

---

## Objectives
- Identify data quality issues and improve data reliability  
- Perform data cleaning and transformation  
- Analyze trends and patterns in project delivery  
- Build KPI-driven insights using Power BI  
- Automate reporting using Excel VBA  
- Maintain version control using Git and GitHub  

---

## Step-by-Step Process

### 1. Data Analysis using SQL
Initial data exploration was performed using SQL to understand the dataset.

- Identified missing values and inconsistencies  
- Analyzed distributions and trends  
- Established a baseline for comparison after cleaning  

---

### 2. Data Extraction and Processing using Python
Python libraries were used for data handling and preprocessing:

- pandas and numpy for data manipulation  
- matplotlib and seaborn for visualization  

#### Activities performed:
- Data extraction from source  
- Handling missing values  
- Data transformation and formatting  
- Exploratory Data Analysis (EDA)  
- Visualizing delivery trends  

---

### 3. Data Storage
After cleaning and preprocessing:

- Cleaned data was stored back into MySQL  
- Also saved as local files for further use  

---

### 4. Data Validation
KPI queries were re-executed on the cleaned dataset to ensure:

- Accuracy of calculations  
- Consistency of data  
- Reliability for business decision-making  

---

### 5. KPI Development using Power BI (DAX)
Dynamic KPIs were created using DAX to measure business performance.

#### Key KPIs:
- On-Time Project Delivery Percentage  
- Resource Utilisation Rate  
- Total Revenue  
- Revenue per Project  
- Average Client Satisfaction Score  
- Employee Utilisation  
- Department Workload  
- Average Delivery Delay  

---

### 6. Power BI Dashboard Development
A multi-page interactive dashboard was developed.

#### Pages included:
- Overview Dashboard  
- Project Analysis  
- Resource Analysis  
- Client Analysis  
- Executive Summary  

#### Features:
- KPI cards for quick insights  
- Bar, line, and pie charts for analysis  
- Conditional formatting for performance tracking  
- Slicers for dynamic filtering  

---

### 7. Excel VBA Automation
An Excel macro was developed to automate summary report generation.

#### Functionality:
- Clears previous report  
- Calculates key metrics  
- Generates summary output  
- Executes with a single button click  

---

### 8. Version Control and GitHub Integration
Git was used for version control and GitHub for project hosting.

#### Steps performed:
- Installed Git  
- Configured user identity (name and email)  
- Initialized local repository using git init  
- Added files using git add .  
- Committed changes using git commit  
- Connected to GitHub repository  
- Pushed code using git push  

---

## Tools and Technologies Used
- SQL (MySQL)  
- Python (pandas, numpy, matplotlib, seaborn)  
- Power BI (DAX)  
- Excel VBA  
- Git and GitHub  

---

## Key Insights
- Project delivery trends highlight delays in certain periods  
- Resource utilization varies across employees  
- A few clients contribute significantly to total revenue  
- Client satisfaction is generally high with some improvement areas  
- Workload distribution varies across departments
